# The Beam brand

This is a place for miscellaneous information and material which can be use to promote **Beam Privacy** (logos, animations, presentations, etc.).

<img src="https://github.com/dbadol/Beam-Material/blob/master/Beam%20Logo%20Pack/Beam_logo_updated.svg" width="25%" height="25%">

## Naming guidelines

The **project name**, shall be written with capital first letter: “**Beam**” 
(not “BEAM”, “beam” or any other variations).

The **currency** shall be written “**beam**” (might be plural: “beams”).<br/>
Example: *Alice sent 1 beam to Bob and she still has 100 beams left.*

The **Currency Code** (“ticker”) is “**BEAM**”
Example: *Bob has 10 BEAM and 1 BTC.*
